# Panduan Menjalankan Aplikasi

## Kloning Repository
```bash
git clone https://github.com/Nasuco/test-fe-ptmauju-annasabdulloh.git
```
## Install Dependensi
1. Masuk ke direktori project yang baru di clone
```bash
cd test-fe-ptmauju-annasabdulloh
```
2. Jalankan perintah
```bash
npm install
```
## Jalankan aplikasi
```bash
npm start
```
